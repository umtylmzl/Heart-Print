 Heart Print — Kolye (Locket) Fotoğraf Yazdırıcı
Kalpli kolye içine koyacağınız fotoğrafı **milimetrik** ölçülerde A4’e yazdırmak için basit bir web aracı.  
Tarayıcıda çalışır; kurulum gerekmez.
## Sorun → Çözüm
Küçük kalp penceresine uygun ölçüde baskı almak zor. Heart Print, fotoğrafı **kalp maskesiyle** önizler, **10 farklı mm boyutu** sunar ve **A4 yazdırmaya** göre stiller.
## Özellikler
- **10 ön tanımlı boyut** (15–25 mm)
- **Kalp kırpma** (SVG `clipPath`) + canlı önizleme
- **Yazdırma**: `@media print`, A4, kesim çizgisi rehberi
- **Responsive** arayüz (mobil / masaüstü)
- **Statik site**: HTML, CSS, JS — build yok
## Teknoloji
| Katman | Teknoloji |
|--------|-----------|
| İşaretleme | HTML5 |
| Stil | Tailwind CSS (CDN) + özel CSS |
| Mantık | Vanilla JavaScript |
| Dağıtım | GitHub Pages uyumlu (statik dosyalar) |
## Kullanım
1. `index.html` dosyasını tarayıcıda açın (veya Pages ile yayınlayın).
2. Fotoğraf yükleyin.
3. Kolye ölçünüze uygun **mm** satırını seçin.
4. **Yazdır** → yazıcıda **ölçek %100** / gerçek boyut, mümkünse arka plan grafikleri açık.
