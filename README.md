💖 Heart Print - Locket Photo Resizer
Heart Print, özellikle kalpli kolye (locket) içine fotoğraf yerleştirmek isteyenlerin yaşadığı "boyut tutturamama" sorununu milimetrik hassasiyetle çözen, web tabanlı bir araçtır.

🚀 Proje Ne İşe Yarar?
Hepimiz o şık, kalpli kolyeleri severiz ancak içine koyacağımız fotoğrafı doğru boyutta bastırmak tam bir kabustur. Çoğu zaman fotoğraf ya çok büyük gelir ya da çok küçük kalarak kolye içinde kaybolur.

Heart Print ile:

Fotoğrafınızı yüklersiniz.

Kolye ucunuzun iç kısmını cetvelle ölçüp uygun şablonu (15mm - 20mm arası) seçersiniz.

Sistem, fotoğrafınızı kalp formuna sokar ve A4 kağıdı üzerinde tam ölçekli bir baskı dosyası hazırlar.

✨ Temel Özellikler
📏 Milimetrik Hassasiyet: 15mm, 16mm, 17mm, 18mm, 19mm ve 20mm gibi en popüler kolye ucu boyutları ön tanımlıdır.

📱 Tam Responsive Tasarım: Mobil cihazlarda kolay kullanım ve önizleme desteği.

🖨️ Baskı Odaklı (Print-Ready): CSS @media print kuralları ile tarayıcı stillerini ezerek doğrudan A4 boyutuna uygun çıktı alma imkanı.

🖼️ Anlık Önizleme: Fotoğrafın kalp içinde nasıl görüneceğini gerçek zamanlı olarak takip edin.

🛠️ Teknik Detaylar
Bu proje modern web teknolojileri ve geliştirici dostu yaklaşımlarla inşa edilmiştir:

Frontend: React / Next.js (Opsiyonel: Kullandığın teknolojiye göre değiştirebilirsin)

Styling: Tailwind CSS (Responsive grid ve dinamik boşluk yönetimi için)

Print Logic: A4 standartlarına (210mm x 297mm) sadık kalarak %100 ölçekli çıktı sistemi.

📖 Nasıl Kullanılır?
Fotoğraf Yükle: Kolyene koymak istediğin anını sisteme yükle.

Boyut Seç: Kolyenin iç genişliğini ölç ve listeden (örn: "Standart 20mm") uygun olanı seç.

Yazdır: "Yazdır" butonuna bas.

Baskı Ayarı: Çıktı alırken yazıcı ayarlarından "Ölçek: %100" (veya "Gerçek Boyut") seçeneğinin seçili olduğundan emin ol.
